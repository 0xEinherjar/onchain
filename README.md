# Onchain Projects

Este repositório contém aplicações Web3 construídas com Vue 3 e Vite.

## Estrutura do Projeto

O repositório é organizado em dois projetos principais:

- **`app`**: Frontend principal da aplicação.
- **`vote`**: Aplicação dedicada para funcionalidades de votação.

## Pré-requisitos

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

## Como Executar

Cada projeto possui suas próprias dependências e scripts. Navegue até o diretório do projeto desejado para começar.

### Projeto `app`

1. Entre no diretório:
   ```bash
   cd app
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

### Projeto `vote`

1. Entre no diretório:
   ```bash
   cd vote
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Tecnologias Utilizadas

- **Vue 3**: Framework JavaScript progressivo.
- **Vite**: Ferramenta de build rápida.
- **Wagmi/Viem**: Interação com blockchain Ethereum/EVM.
- **Pinia**: Gerenciamento de estado.
